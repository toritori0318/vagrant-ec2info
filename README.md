#  vagrant-ec2info

Show the ec2 metadata from vagrant. (required vagrant-aws)

## Show Items

- Instance ID
- Instance Type
- Image ID
- Region
- Availability Zone
- Keypair
- Security Groups
- Private IP
- Private DNS
- Public IP
- Public DNS
- Status
- Launch Time
- Tags
- Vpc ID
- Subnet ID

## Usage

    vagrant ec2info <vm_name>
